# Hotel & Flight Booking Platform

A full-stack production-ready booking platform for hotels and flights built with Next.js and NestJS.

## Tech Stack

- **Frontend**: Next.js (App Router), TypeScript, Tailwind CSS, Redux Toolkit
- **Backend**: NestJS, TypeScript, MongoDB with Mongoose
- **Authentication**: JWT with social login support

## Project Structure

```
hotel-flight-booking/
├── frontend/          # Next.js frontend application
├── backend/           # NestJS backend application
└── docs/              # Documentation
```

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- MongoDB (local or Atlas)

### Installation

1. Clone the repository
2. Install dependencies for both frontend and backend
3. Set up environment variables
4. Run the development servers

## Features

- Flight and hotel search
- User authentication (email/password and social login)
- Booking management
- User dashboard
